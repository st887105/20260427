1 帳號與環境準備

① Google 帳號設定

我們需要 Google 雲端硬碟來存放資料庫 (Google 試算表) 和後端應用程式 (Google Apps Script)。

關於帳號選擇：

✅ 可以使用學校教育版帳號，但需配合後續步驟的 Cloudflare Workers 來解決跨域 (CORS) 權限阻擋問題。

✅ 若無特別需求，建議使用個人的 @gmail.com 帳號，設定上最為簡便直接。

② 註冊 GitHub 帳號

GitHub 是存放前端網頁檔案並提供免費網頁代管服務的平台。

  • 前往 GitHub 註冊頁面。
  • 輸入 Email，設定密碼與使用者名稱。
  • 至信箱收取驗證碼，完成註冊流程。

3 建造雲端大腦 (GAS 設定)

3-1 後端參數與對應設定

已鎖定之後端 (Code.gs) 模組

系統已根據您在「前端 (UI)」的選擇，自動綁定對應的後端程式碼。

建立雲端硬碟資料夾與取得 ID

若專案涉及檔案存取，系統需要您提供資料夾 ID。

範例網址：
drive.google.com/drive/folders/1A2B3C_xyz9876543210

☝️ 進入雲端資料夾後，網址中最後面的一串英文數字就是您的 ID。

務必對該資料夾開啟「知道連結的人皆可編輯或檢視」權限。
